# 🚀 Open Source Project Manager (Python CLI)

A simple command-line Python application to simulate and manage an open-source project.
It allows users to:

* Create project metadata
* Add contributors
* Track and analyze issues
* Generate reports (TXT + CSV)
* Perform basic data analysis

---

## 📌 Features

* 📦 Project initialization (name, version, year, language, lead)
* 👨‍💻 Contributor management (list, sorting, status updates)
* 🐞 Issue tracking system (priority, status, reporter)
* 📊 Data analysis:

  * Open issue count
  * Priority breakdown
  * Top reporter detection
* 📁 File handling:

  * Auto folder creation
  * Text report generation
  * CSV export
* ⚡ Bonus:

  * Urgent issue detection (Critical / High)

---

## 🛠️ Requirements

* Python **3.8 or higher**

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the folder:

   ```bash
   cd your-repo-name
   ```

3. Run the script:

   ```bash
   python task5.py
   ```

---

## 📂 Output Files

After execution, a folder will be created:

```
project_name/
│── project_report.txt
│── issues.csv
```

---

## 🧠 Concepts Used

* Tuples (immutable project data)
* Lists & Dictionaries
* Sets & Set operations
* File handling (read/write/append)
* Exception handling
* Basic data analysis logic
